Description de l'application pour le PWM qui reprend l'utilisation des 2 périphériques : ADC et le TIMER cofiguré PWM 
Objectif : Lire l'entrée ADC provenant du potentiomètre qui nous permet d'avoir différentes valeurs à partir desquelles on fixe la valeur du rapport cyclique (duty cycle) du PWM configuré sur le Pin 8 
Câblage : Pin 4 ==> Potentionmètre
	  Pin 8 ==> Visualision su l'oscilloscope (lié à la sonde)
